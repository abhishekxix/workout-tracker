# Workout Tracker Backend

## Install the project

`npm install`

## Setup the following .env variables

- `PORT`
- `MONGO_URI`
- `SENDER_EMAIL`
- `SENDER_PASSWORD`
- `JWT_SECRET`
- `JWT_VERIFICATION_LIFETIME`
- `COOKIE_SECRET`
- `AUTH_TOKEN_LIFETIME`

To connect to a local mongodb instance set `MONGO_URI=mongodb://127.0.0.1:27017/workout-app`

## Build the code

`npm run build`

## Start the server

`npm start`

## Start the server in dev mode

`npm run dev`
